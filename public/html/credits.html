Dependencies

Dependencies

1. axios: HTTP client for making requests.
2. bcrypt: Library for hashing passwords.
3. cookie-parser: Middleware for parsing cookies.
4. cors: Middleware for enabling Cross-Origin Resource Sharing.
5. date-fns: Utility library for date manipulation.
6. dotenv: Loads environment variables from a .env file.
7. express: Web framework for Node.js.
8. joi: Schema validation library.
9. jsonwebtoken: Library for working with JSON Web Tokens.
10. mockingoose: Mocking library for Mongoose models.
11. mssql: SQL Server client for Node.js.
12. swagger-autogen: Automatic Swagger documentation generation.
13. swagger-ui-express: Middleware for serving Swagger UI.
14. uuid: Library for generating unique IDs.

DevDependencies

1. @shelf/jest-mongodb: Jest integration for MongoDB.
2. jest: Testing framework.
3. mongodb-memory-server: In-memory MongoDB server for testing.
4. mongoose: ODM (Object Data Modelling) library for MongoDB.
5. nodemon: Tool for automatically restarting the application.
6. supertest: HTTP assertions library for testing Express applications.
